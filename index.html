Prework Study Guide

HTML

  • The title element defines the title of the webpage and is important for a search engine optimization (SEO, i.e., Google or Bing).
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document

CSS

  • When a developer wants to style an element directly in the HTML file, they use inline CSS. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. An example of inline CSS is using the “style” attribute to change the color of a singe “h1” heading.
  • It is also possible to embed an entire stylesheet directly in the HTML file using a style element in the head element. This internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • An external CSS style sheet is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier. When using an external style sheet, we link the file using a link element placed in the HTMLs head.
  • Separation of concerns is a widely used term in web development and refers to separating code into different files based on use.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property. The declaration “color: blue;” includes the CSS property “color” and the value for that property, “blue.”
  • A link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • A class attribute adds styles to certain elements, but not all elements. They allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A “margin” property acts much like a margin in a word processing document, it indicates how much space we want around the outside of an element. When it is set to 0, we are indicating that we want NO extra space around the element.
  • The “padding” property is often confused with “margin,” but it is not the same thing. It adds space around the content inside an element. When you increase padding, you are making the entire element bigger.
